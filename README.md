# srinivasanbotree.github.io

GitHub Pages: main site and project index.

## Description

Index of GitHub Pages repositories: project pages, documentation and portfolio. Browse and open individual project sites from a single main page.

## Site

- [Open site](https://srinivasanbotree.github.io/)
- [Sitemap](https://srinivasanbotree.github.io/sitemap.xml)

## Instructions

1. In the project folder add clone URLs to **list.txt** (step 4 in the terminal).
2. Run **Run REPO.bat** — repos are cloned; Google verification tag is added to each index.html if uploaded.
3. Run **Run MAIN.bat** — main site (index and sitemap) is updated.
